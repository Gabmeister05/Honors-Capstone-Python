# Honors-Capstone-Python
A collection of Python-based data analysis/automation projects (the Capstone project for UMich), featuring BeautifulSoup, Requests, and SQLite.

## Project Overview
This project is the final Honors Capstone for the **Python for Everybody Specialization** by the University of Michigan. 
The goal was to build a complete data pipeline: from retrieving raw data from the web to processing it through a database and generating meaningful visualizations.

I successfully completed this project with **Honors**, demonstrating proficiency in data structures, web scraping, and database management.

## Key Features
* **Data Retrieval:** Automated scraping and fetching of data from Google API.
* **Data Processing:** Cleaned and structured raw JSON/XML data for analysis.
* **Database Management:** Utilized **SQLite** to store and query large datasets efficiently.
* **Visualization:** Developed interactive and static visualizations using Matplotlib to reveal patterns and insights.

## Tech Stack
* **Language:** Python 3.x
* **Libraries:** [e.g., SQLite3, BeautifulSoup4, Urllib, Matplotlib]
* **Data Formats:** JSON, XML, SQL
